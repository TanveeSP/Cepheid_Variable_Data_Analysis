## 📘 Problem Overview

This project focuses on the analysis of **Cepheid Variable Stars**, which play a fundamental role in determining astronomical distances.

Cepheid variables are pulsating stars whose brightness varies periodically. A key property of these stars is the **Period–Luminosity Relation**, which establishes a direct connection between the pulsation period and intrinsic luminosity.

Using this relation, astronomers can estimate distances to faraway galaxies, making Cepheids one of the most important tools in observational astrophysics.
